# Матрица сценариев продуктового бота

Кратко: вход → действие → ожидаемый результат.

## Онбординг

| Шаг | Действие | Результат |
|-----|----------|-----------|
| Первый /start | Пользователь новый | Три экрана подсказок или «Пропустить», флаг онбординга сохраняется в привязке бота |
| Повторный /start | Онбординг уже пройден | Приветствие и главное меню |

## Главное меню

| Кнопка | Результат |
|--------|-----------|
| Агенты | Список агентов, политика автоответов, создание агента |
| Заметки | Список заметок, добавление заметки / напоминания по шагам |
| Новости | Подписки, сбор дайджеста |
| Помощь | Текст HELP_TEXT |
| Режим чатов | Список диалогов MTProto → чат: режим per-dialog, сброс, назначение агента |

## Режим чатов (без TgAccount)

| Действие | Ответ бота |
|----------|------------|
| Открыть «Режим чатов» | Сообщение о необходимости подключить личный аккаунт |

## Режим чатов (с TgAccount)

| Действие | Результат |
|----------|-----------|
| Выбрать чат | Панель: manual / suggest / auto, сброс на глобальный, назначить агента |
| Сброс | Удаляется `dialogOverrides[dialogId]` в политике |
| Назначить агента | Создаётся/обновляется `ProductAgentDialog` |

## Напоминания (callback)

| callback_data | Условие | Эффект |
|---------------|---------|--------|
| rem:ok:&lt;id&gt; | Напоминание принадлежит пользователю | status = completed |
| rem:zz:&lt;id&gt; | status = awaiting_confirm | fireAt += snooze, status = pending |

## Автоответ (MTProto)

| Условие | Job не создаётся |
|---------|------------------|
| effectiveReplyMode manual для диалога | Да (подсказка в логах) |
| allowlist и чата нет в списке | Да |
