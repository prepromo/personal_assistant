# Ручной чеклист MVP (после доработок бота)

- [ ] `npx tsc --noEmit` в каталоге `telegram-user` без ошибок
- [ ] `GET http://127.0.0.1:4050/health` — `ok: true`, при настроенном LLM — `llm.configured: true`
- [ ] `/start` — онбординг (новая привязка) или меню (старая)
- [ ] **Агенты** → политика открывается при наличии TgAccount
- [ ] **Заметки** → заметка и напоминание по шагам
- [ ] **Новости** — хаб открывается
- [ ] **Помощь** — текст без «сырых» ошибок
- [ ] **Режим чатов** — список или сообщение о подключении аккаунта; при списке — смена manual/suggest/auto и назначение агента
- [ ] Напоминание с подтверждением: кнопки **Сделано** / **Ещё нет** (см. `REMINDER_SNOOZE_MINUTES`)

**Вне проверки этого чеклиста:** деплой в РФ, оплата, прод-webhook.
