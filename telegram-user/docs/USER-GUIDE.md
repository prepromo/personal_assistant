# Руководство пользователя (продуктовый бот)

Сервис состоит из **бота в Telegram** (меню и напоминания) и **личного аккаунта Telegram**, с которого при необходимости уходят автоответы. Управлять «плавающей кнопкой» внутри каждого чата Telegram **нельзя** — так устроен клиент; все настройки делаются **в этом боте**.

## Первый запуск

1. Откройте бота и отправьте `/start`.
2. Пройдите короткий онбординг (**Далее** / **Пропустить**) или пропустите его.
3. Чтобы работали автоответы в обычных переписках, один раз подключите личный аккаунт через **коннектор** (вход по коду из Telegram). `APP_USER_ID` в настройках коннектора должен совпадать с привязкой вашего профиля.

## Кнопки меню

| Кнопка | Назначение |
|--------|------------|
| **Агенты** | До 5 ассистентов с разными инструкциями; кнопка **Политика и автоответы** — глобальный режим (manual / suggest / auto), группы, список чатов для агента. |
| **Заметки** | Текстовые заметки и пошаговое **Напоминание**. Встречи из переписки могут сохраняться автоматически (если включены LLM и автосохранение на сервере). |
| **Новости** | Подписки на диалоги MTProto или чаты бота, сбор дайджеста. |
| **Помощь** | Краткая справка. |
| **Режим чатов** | Для **каждого диалога** отдельно: manual, suggest или auto (перекрывает общий режим). **Назначить агента**, **Написать первым** (очередь на отправку с личного аккаунта через worker). |

### Быстрые ссылки и фразы

- В карточке чата (**Режим чатов** → чат) при заданном в настройках сервера `PRODUCT_BOT_USERNAME` показываются ссылки **t.me/…?start=…**: открывают бота и сразу выставляют режим для этого чата.
- Если включено `PRODUCT_BOT_NL=1`, свободный текст может распознаваться как **команда**; перед выполнением — **Подтвердить** / **Отмена**. Обычный свободный чат с LLM **не выполняет** действия в базе — только подсказки.
- Напоминание через «Заметки» после ввода минут спрашивает подтверждение. Добавление агента по имени — тоже с подтверждением.

## Напоминания с подтверждением

Напоминания, созданные из автоответов о встрече, приходят в **личку с ботом** с кнопками:

- **Сделано** — напоминание закрыто.
- **Ещё нет** — отложить на время из настройки сервера (`REMINDER_SNOOZE_MINUTES`, по умолчанию 60 минут).

## Кабинет и API

Веб-кабинет на том же сервисе (`cabinet.html`) дублирует часть настроек. Деплой на свой сервер и оплата — отдельный этап, не входят в локальный MVP.
