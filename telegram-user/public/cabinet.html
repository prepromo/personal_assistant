Личный кабинет

На главную (лендинг)

Три шага до полного доступа

  1. Сайт — регистрация. Email и пароль (ниже) или «Вход через Telegram». Система сама выдаёт appUserId. После регистрации на сайте — 1 день бесплатного доступа (подключение личного Telegram и функции кабинета).
  2. Сайт — тариф. После триала — 500 ₽ за месяц. Оплата через ЮKassa / ЮMoney (кнопка в блоке «Статус»). На стенде без ЮKassa — тестовая кнопка при BILLING_ALLOW_SIMULATED_PAYMENT=1. Досрочная оплата во время триала тоже возможна.
  3. Личный Telegram. В кабинете блок «Личный Telegram» или в боте: номер +7… → код из Telegram → при необходимости пароль 2FA. На сервере должен работать worker — тогда появятся диалоги и автоответы.

Док: USER-TURN, AUTOMATION. В @BotFather: /setdomain для виджета входа.

Шаг 1 — регистрация

Сначала аккаунт на сайте (есть 1 день триала), затем при необходимости оплата (шаг 2), затем подключение личного Telegram (шаг 3).

Вход через Telegram — подтверждение в приложении, без пароля.

Регистрация по email — создаётся appUserId автоматически, коннектор подключаете после оплаты.

Расширенно: у меня уже есть appUserId из коннектора

Если TgAccount уже создан через ensure-account, можно привязать кабинет к этому id.

Вход через OIDC (нужны OIDC_* в .env; appUserId — из поля выше)

Личный Telegram

Подключение как в обычном входе: номер +7… → код в Telegram → при необходимости пароль 2FA. Нужны TELEGRAM_API_ID / TELEGRAM_API_HASH в connector/.env на сервере.

Нужны запущенные API 4050, connector worker, OPENCLAW_* (или OpenAI) в .env. Сначала suggest, затем auto. Док: AUTOMATION

В режиме списка отметьте чекбоксы у чатов и нажмите «Сохранить список». Кабинет и ручная отправка не ограничиваются.
