# Glassmorphism (Comrade UI)

Design direction aligned with [typeui.sh — Glassmorphism](https://www.typeui.sh/design-skills).  
To pull the official skill file later: `npx typeui.sh pull glassmorphism`

## Tokens

- **Background:** deep slate gradient `#0f172a` → `#1e1b4b`
- **Glass surface:** `bg-white/5` + `backdrop-blur-xl` + `border border-white/10`
- **Accent:** cyan `#22d3ee` / violet `#a78bfa`
- **Text:** `text-slate-100` / muted `text-slate-400`
- **Radius:** `rounded-2xl` cards, `rounded-xl` buttons
- **Shadow:** soft glow on focus/hover
