Каналы связи (Telegram)

На openclaw.ru каналы подключаются в облаке. У вас локально роль канала выполняет OpenClaw gateway с токеном бота из openclaw/.env.

Шаги

  1. Создайте бота в @BotFather, получите TELEGRAM_BOT_TOKEN.
  2. Пропишите токен в openclaw/.env и запустите openclaw\scripts\start-gateway.ps1 (или полный стек из README OpenClaw).
  3. При dmPolicy: pairing одобрите доступ: openclaw pairing approve telegram …
  4. Этот веб-портал (local-saas) — отдельно; чат в браузере идёт через gpt2giga или через gateway, если выбран провайдер OpenClaw в кабинете.

Личный аккаунт Telegram (не бот)

Для MTProto и синка диалогов используйте проект telegram-user на порту 4050 — см. telegram-user/README.md.
